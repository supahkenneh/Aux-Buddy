# **Aux Buddy**

A web app that interacts with the Spotify API which allows users to create playlists on the fly by inputting their favorite artists and songs. 

**A Spotify account is required to use the web app**

### Technologies used
- React
- Node.js
- Express.js
- Spotify API
- Tailwind CSS
